## brew安装所需要的常用插件

```bash
$ brew install qlmarkdown			#makedown 显示插件
$ brew install qlcolorcode		#快速查看代码 并赋予格式和颜色的插件
	$ brew reinstall --cask qlcolorcode
$ brew install qlstephen      #快速查看大部分格式的文本文件插件
	$ brew reinstall --cask qlstephen    #快速查看大部分格式的文本文件插件
$ brew install highlight luarocks   quicklook-json	#快速查看json

```
